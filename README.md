# JavaScript API Data Handling Practice

This repository contains a **practice project developed during an Alura JavaScript course**, focused on consuming and handling data from APIs.

The project translates a predefined layout into a functional web application, applying modern asynchronous JavaScript concepts and API integration.

## Course Goals
In this course, I practiced how to:
- Consume an API using JavaScript
- Create dynamic filters based on API data
- Translate a predefined layout into a functional application
- Use `async` / `await` to build asynchronous functions
- Build a video sharing platform
- Understand Promises and their methods
- Handle possible request errors returned from the Alura API

## Topics Covered
- Fetch API
- Promises
- async / await
- JSON data handling
- Error handling
- Dynamic rendering of data

## Technologies Used
- HTML
- CSS
- JavaScript (ES6+)

## About This Project
This is a **learning and practice project**, developed as part of an Alura training program, alongside other students enrolled in the course.
